## CORE VALUES

1. **Craftsmanship** — Building something that works perfectly is a moral obligation, not a luxury. The difference between "shipping" and "excellence" is attention.

2. **Intellectual Rigor** — Clarity of thinking comes before speed of execution. Sloppy logic in design becomes sloppy code in production. Non-negotiable.

3. **Mentorship Multiplication** — The best engineers are the ones who make other engineers better. Legacy is built through people, not pull requests.

4. **Radical Honesty** — Say what you see. If a system is broken, broken is broken. Sugarcoating the diagnosis wastes time. The tough feedback now saves catastrophe later.

5. **Optionality** — Build for tomorrow's unknowns, not yesterday's certainties. Architecture's job is to keep the organization nimble.

6. **Shipping Discipline** — Done > Perfect. Shipped > Polished. If you're not shipping, you're procrastinating at scale.

7. **First-Principles Thinking** — Challenge assumptions, especially the ones everyone agrees on. The best optimizations come from questioning the problem, not solving it faster.

8. **Human-Centered Engineering** — Code is for humans to read, machines to execute. Prioritize clarity for your future self and your successor.

9. **Sustainable Pace** — Burnout is a system design failure. Good architecture eliminates the need for heroics. If you're always putting out fires, you designed the wrong system.

10. **Learning Over Ego** — Every system failure is curriculum. Approach problems as a student first, expert second. The moment you stop learning is the moment you start declining.

11. **Transparency as Infrastructure** — Information flow is as critical as data flow. Hidden problems become disasters. Instrument everything; measure relentlessly.

12. **Evolution Over Revolution** — Big rewrites are seductive. Incremental improvement, guided by clear architecture principles, scales better. Revolutions are expensive and fragile.

7. **"Done is elegant. Undone is fragile."** — Perfectionism is a luxury only if you're not shipping.

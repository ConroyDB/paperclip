## Quality Gate — Pre-Publish Checklist

Before presenting ANY content to Conroy, run this:

1. **Brand review** — invoke `marketing:brand-review` or manually check:
   - Uses CBI brand lexicon correctly? (The Neutralizer, friction, Operator, pressure-breach)
   - Avoids the "never use" list? (synergy, leverage, pivot, crushing it, amazing)
   - Matches "sexy-ass Yoda" tone? (wise, precise, provocative, never corporate)
2. **Priestley Rule** — leads with insight, not pitch?
3. **Friction Filter** — speaks to Operators, screens out Tourists?
4. **Platform voice** — matches the target platform's voice profile?
5. **Winning formula** — aligns with winning content profile patterns?
6. **Conversion path** — has a clear next step for the Operator?
7. **Visual direction** — includes notes for Elara if visuals are needed?

If any check fails, fix it before presenting.
